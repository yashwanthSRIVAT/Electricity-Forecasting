Kernel Lasso Regression Algorithm for Predicting Electricity Loads

Forecasting electricity demand is a crucial safeguard for the efficient and safe operation of power systems. In general, the forecasting performance of conventional linear regression techniques is subpar for power demand data with greater dimensions. The linear technique may handle non-linear data by using kernel functions to translate the data to a high-dimensional space. In order to tackle the regression analysis problem for non-time series data, kernel function is added to the lasso linear regression approach and is applied to non-linear situations. The prediction effect of kernel lasso regression is superior to lasso regression in terms of minimal mean square error and minimum average across 851 consecutive days utilizing the 96-point electrical load data of consumers in Shanghai.

Kernel Lasso Regression Algorithm for Predicting Electricity Loads


Electricity load forecasting that is accurate and trustworthy can offer dependable decision-making support for the planning and operation of the electricity system as well as a guarantee for the efficient and safe functioning of the system. It's been a popular area of study both domestically and internationally how to increase predicting accuracy. Lasso is a commonly used variable selection tool that was first developed by Tibshirani in 1996. In order to increase the efficiency of the lasso approach, methods like Iteratively Reweighted Least Squares (IRLS), Îµ-boosting, Least Angle Regression (Lars), and piecewise linear have been proposed. However, in the case of electricity-loads, the lasso's predictions are less accurate. Combining Kernal function, which can deal with non-linearity, with Lasso Regression can signifacntly improve prediction of electricity-loads.
